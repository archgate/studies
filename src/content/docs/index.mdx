## Mission

Archgate Studies publishes **reproducible, peer-reviewable research** on how teams govern their software architectures in practice. Every study ships its method, assumptions, code, data artifacts, and interpretation so that results can be independently verified and extended.

<CardGrid>
  <Card title="Reproducible" icon="document">
    Every study includes scripts, raw data, and methodology documentation so
    results can be independently verified.
  </Card>
  <Card title="Evidence-Based" icon="magnifier">
    Findings are grounded in real-world repository data — pull request activity,
    review cycles, and merge latency from open-source projects.
  </Card>
  <Card title="Actionable" icon="rocket">
    Each study concludes with concrete Architecture Decision Record (ADR)
    proposals that teams can adopt directly.
  </Card>
  <Card title="Open" icon="github">
    All code, data, and analysis live in a public GitHub repository. Peer review
    and contributions are welcome.
  </Card>
</CardGrid>
